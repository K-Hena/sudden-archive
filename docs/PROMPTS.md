# 공통 작업 원칙

모든 작업에서 반드시 지켜야 할 사항

- AI_CONTEXT.md를 기준으로 작업한다.
- 기존 구조를 최대한 유지한다.
- 추측하지 않는다.
- 존재하지 않는 파일을 가정하지 않는다.
- 필요한 파일만 수정한다.
- 완료 후 영향 범위를 확인한다.
- 사용자 승인 없이 구조를 변경하지 않는다.

↓

↓

↓

↓

배포 확인

Claude Code에게 전달하는 작업 지시서(코드 작업)는 끝에 항상 "작업 완료 후 git add / commit / push까지 진행"을 포함한다.
